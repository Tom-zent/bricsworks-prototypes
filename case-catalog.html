세부기획서 §5-2(검증 규칙)·§5-3(검증 결과별 동작)·§5-5(기존 입력분 처리)에 정의된 모든 상태를 케이스별로 정리했다. 각 케이스는 발생 조건 → 시스템 동작 → 노출 문구 → 해당 상태의 UI 순으로 구성된다. 마지막 절에서 실제로 조작해볼 수 있다.
원칙: ⑯ 국세환급금 계좌신고 행은 직접 수정 불가(읽기 전용)이며 모든 입력·수정은 [계좌입력] 모달을 통해서만 이뤄진다.
