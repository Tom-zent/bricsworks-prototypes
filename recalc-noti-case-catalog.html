재계산 미실행 노티 — 화면 상태 카탈로그

화면 톤(색상·버튼·단계 스텝퍼)은 브릭스웍스 실제 화면 기준으로 통일했고, 알림 카드는 기존에 쓰던 "선행 단계 데이터 변경" 알림과 같은 스타일을 재사용했습니다. 케이스 00에서 기존 패턴과 신규 적용을 나란히 비교합니다.

본검토 단계의 원천 데이터가 2026.07.16 18:09에 갱신되었으나, 현재 작성 데이터는 2026.07.16 17:29 기준입니다. 재계산을 실행해 주세요.

신규 — 전자신고 [검증하기] 클릭 시에만 노출
⚠︎

재계산이 실행되지 않았습니다

작성 단계의 데이터가 수정되었으나, 재계산이 아직 실행되지 않았습니다. 그래도 진행하시려면 아래 버튼을 눌러주세요.

기존 알림은 단계 진입 시 자동으로 뜨는 안내형(액션 버튼 없음)이고, 이번 건은 전자신고 단계의 검증하기 클릭이라는 명확한 시점에만 뜨며 [그래도 검증 진행] 액션이 추가된다는 점이 유일한 차이입니다. 색상·아이콘·카드 구조는 동일하게 맞췄습니다.
